# xlake-demo
A High-Performance, Distributed Data Lake Platform with Database-Like Simplicity

**Xlake** is an open-source, high-performance distributed data lake platform designed to provide **database-like simplicity** for big data workloads. Built with modern Java 21 and integrated with Apache Spark, Xlake offers:

- **Unified Data Management**: Manage structured and semi-structured data with SQL-like interfaces
- **High Availability**: Raft-based consensus algorithm ensures data consistency and fault tolerance
- **Multi-Backend Support**: Seamless integration with HDFS, S3, OSS, and local file systems
- **ACID Transactions**: Full support for snapshot isolation and atomic commits
- **Schema Evolution**: Flexible schema management with versioning and migration support
- **Vectorized Execution**: Optimized query processing with columnar storage formats (Parquet, ORC，Lance etc)
